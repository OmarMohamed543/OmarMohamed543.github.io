Card Search Engine

Languages: Pugjs CSS JavaScript Expressjs Dockerfile

Programs: GitHub Gitbash Visual Studio Code

Project Website: Link to the project
GitHub Repository: Link to the GitHub repository

The purpose of this website is to serve as a user-friendly search engine for Disney's TCG Lorcana cards. It utilizes the Lorcana API to offer users a one-stop destination for exploring cards within the Lorcana sets, complete with comprehensive details.

This innovative card search engine project seamlessly combines responsive design and full mobile compatibility, delivering a visually captivating user experience. Users benefit from case-insensitive search functionality and an array of advanced features, simplifying the exploration and discovery of an extensive card collection. The platform's clean, intuitive design, coupled with lightning-fast loading times, ensures your card searches are not only efficient but also aesthetically engaging. Accessibility takes precedence, making it universally user-friendly. With cross-browser compatibility, every user enjoys a consistently smooth experience, ensuring the project's accessibility and appeal to a wide audience.
